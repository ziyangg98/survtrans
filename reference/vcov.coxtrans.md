# Variance-covariance matrix for a `coxtrans` object.

Variance-covariance matrix for a `coxtrans` object.

## Usage

``` r
# S3 method for class 'coxtrans'
vcov(object, ...)
```

## Arguments

- object:

  An object of class `coxtrans`.

- ...:

  Additional arguments (unused).

## Value

A matrix representing the variance-covariance matrix of the
coefficients.
