# Variance-covariance matrix for a `ncvcox` object.

Variance-covariance matrix for a `ncvcox` object.

## Usage

``` r
# S3 method for class 'ncvcox'
vcov(object, ...)
```

## Arguments

- object:

  An object of class `ncvcox`.

- ...:

  Additional arguments (unused).

## Value

A matrix representing the variance-covariance matrix of the
coefficients.
