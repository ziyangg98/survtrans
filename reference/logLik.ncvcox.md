# Log-likelihood for a `ncvcox` object

Log-likelihood for a `ncvcox` object

## Usage

``` r
# S3 method for class 'ncvcox'
logLik(object, ...)
```

## Arguments

- object:

  An object of class `ncvcox`.

- ...:

  Additional arguments (unused).

## Value

A numeric value representing the log-likelihood of the fitted `ncvcox`
object.
