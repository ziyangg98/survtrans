# Log-likelihood for a `coxtrans` object

Log-likelihood for a `coxtrans` object

## Usage

``` r
# S3 method for class 'coxtrans'
logLik(object, ...)
```

## Arguments

- object:

  An object of class `coxtrans`.

- ...:

  Additional arguments (unused).

## Value

A numeric value representing the log-likelihood of the fitted `coxtrans`
object.
