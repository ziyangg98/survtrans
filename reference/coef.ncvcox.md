# Extract the coefficients from a `ncvcox` object

Extract the coefficients from a `ncvcox` object

## Usage

``` r
# S3 method for class 'ncvcox'
coef(object, ...)
```

## Arguments

- object:

  An object of class `ncvcox`.

- ...:

  Additional arguments (unused).

## Value

A named numeric vector containing the coefficients of the fitted
`ncvcox` object. Zero coefficients are removed.
